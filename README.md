"# Pseudo-Telegram-Bot" 
- A simple Telegram bot that has a mini web built using Node.js.

## Installation
- 1. Clone the repository
- 2. Run 'npm install'
- 3. Set environment variables

## Usage
- 1. Start the bot using 'node index.js'
- 2. Interact with the bot using Telegram
